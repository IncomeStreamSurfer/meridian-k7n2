# Meridian — Specialty Coffee, Coming Soon

A coming-soon landing experience for **Meridian**, a specialty coffee brand.
Editorial dark/light aesthetic, email capture wired to Supabase, and three
tight pages: Home, Our Story, and a thank-you confirmation after signup.

## Stack

- **Astro 5** (server output) with the Vercel adapter
- **Tailwind v4** via `@tailwindcss/vite`
- **Supabase** (`meridian_waitlist`, `meridian_content` tables)
- **Resend** — welcome email via onboarding@resend.dev
- **@astrojs/sitemap** — auto-generates `/sitemap-index.xml`

## Pages

| Path       | Purpose                                              |
| ---------- | ---------------------------------------------------- |
| `/`        | Coming-soon hero + brand story + waitlist capture    |
| `/about`   | Long-form "Our Story" — sourcing, roast, values      |
| `/thanks`  | Post-signup confirmation + share prompts             |
| `/robots.txt` | Generated dynamically, references the sitemap     |
| `/sitemap-index.xml` | Auto-generated by `@astrojs/sitemap`        |

## API

- `POST /api/subscribe` — accepts `{ email }` (JSON or form).
  - Validates email
  - Inserts into `meridian_waitlist` (handles duplicates gracefully)
  - Fires a Resend welcome email
  - Returns JSON `{ ok: true }` or `{ error }`

## Environment


## Supabase schema

The `meridian_waitlist` table:

| column     | type                     |
| ---------- | ------------------------ |
| id         | uuid (pk)                |
| email      | text (unique)            |
| source     | text                     |
| referrer   | text                     |
| created_at | timestamptz              |

The `meridian_content` table is reserved for the Harbor Writer to publish
articles into later.
